# IBM-HR-Analytics
This Project analyzes IBM's HR dataset to uncover information about employee attrition, age distribution, job satisfaction, and other HR trends.
## Tools Used
Microsoft Excel(Pivot Tables, Charts)
## Key Insights
Identified major factors influencing employee attrition, visualized trends by age, department, and job role, and developed a dashboard for HR decision-making
## Dataset 
IBM HR Dataset [Kaggle] (https://www.kaggle.com/pavansubhasht/ibm-hr-analytics-attrition-dataset)
